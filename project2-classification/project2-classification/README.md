# Project 2 — Iris Classifier 🌸
ML Classification using K-Nearest Neighbors.

## How to Run
pip install scikit-learn pandas numpy
python iris_classifier.py

## Pipeline
1. Load Iris Dataset (150 samples)
2. Feature Scaling (StandardScaler)
3. Train-Test Split (100 train / 50 test)
4. Train KNN Model (K=5)
5. Evaluate (Confusion Matrix + F1 Score)

## Skills
Scikit-learn | KNN | Feature Scaling | Model Evaluation
