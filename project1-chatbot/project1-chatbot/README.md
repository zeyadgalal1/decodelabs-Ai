# Project 1 — ZeyadBot 🤖
Rule-Based AI Chatbot built with Python.

## How to Run
python zeyadbot.py

## Features
- Dictionary-based responses (O(1) lookup)
- Input sanitization (.lower().strip())
- Continuous conversation loop (while True)
- Fallback for unknown inputs (.get())
- Clean exit strategy (break)

## Skills
Control Flow | Dictionary Lookup | Basic AI Concepts
